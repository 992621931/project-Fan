卡牌预制体元素
id (string) - 卡牌唯一标识符

例如: "card_liangzi"
name (string) - 卡牌名称

例如: "凉子"
rarity (string) - 稀有度

可选值: common（普通）、rare（稀有）、mythic（神话）、legendary（传说）
例如: "rare"
image (string) - 卡牌图片路径

例如: "images/kapai_liangzi.png"
width (number) - 卡牌宽度（像素）

例如: 300
height (number) - 卡牌高度（像素）

例如: 500
holographic (boolean) - 是否有闪膜效果（数据层标记，实际效果由代码控制）

例如: false
obtainMethod (string) - 获取方式

例如: "无"
description (string) - 卡牌描述

例如: "酒馆的女仆店员，每周一和周二会在酒馆工作，温柔可爱，深受客人们的喜爱。"
